## Objetivo y Descripción del proyecto

Este repositorio contiene el trabajo realizado por el grupo Abejorros para la asignatura de Sistemas de Apoyo a la Decisión, donde simulamos ser un equipo de analistas que trabaja para una app de citas, en nuestro caso, Bumble.
Como resultado, debemos realizar un analisis de las distintas valoraciones y sacar conclusiones de ellas. Realizando un estudio de la situacion de la empresa a lo largo del tiempo, además de tener en cuenta otro tipo de variables.

---

## Contenido del Proyecto

* **`clasificacion`**: Generación de modelos de clasificación tradicionales.
* **`clustering`**: Generación de clusters sobre los diferentes sentimientos identificados.
* **`datos`**: Datos necesarios para el estudio.
* **`generacion`**: Clasificación de sentimientos y oversampling mediante modelos generativos.
* **`requirements.txt`**: requirements general.

Se recomienda entrar en cada apartado para ejectuar y visualizar los resultado necesarios. Cada sección tiene su propio README y requirements.txt detallando exactamente el funcionamiento y especificaciones necesarias para el correcto uso.
